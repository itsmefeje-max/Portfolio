# Arcade Modules

This directory contains the JavaScript modules for the Live Orbital Arcade simulation.

- `TimeEngine.js`: Manages UTC time and astronomical calculations.
- `GeoService.js`: Handles IP geolocation.
- `ISSService.js`: Fetches and interpolates ISS position.
- `EarthRenderer.js`: Three.js scene setup and rendering.
- `CameraDirector.js`: Manages camera modes.
- `HUDLayer.js`: Manages the UI overlay.
- `main.js`: Entry point.
